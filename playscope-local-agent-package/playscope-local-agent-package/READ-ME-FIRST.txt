PlayScope Local Agent

Bu küçük program PlayScope için lokal mouse kontrolü yapar.

Ne yapar:
- Render veya lokal PlayScope websitesinden gelen isteği dinler.
- YouTube linkini açar.
- Mouse'u varsayılan olarak ekranın sağ üstüne götürüp tıklar. Bu Scrumball extension/panel açmak içindir.
- Scrumball panelini kırpar.
- Görüntüyü PlayScope websitesine geri verir.

Çalıştırma:
1. start-playscope-local-agent.bat dosyasına çift tıkla.
2. Pencere açık kalsın.
3. Render'daki PlayScope'ta Influencer Bul > YouTube + Scrumball auto OCR alanını kullan.

Ayar gerekirse:
- clickXPercent: varsayılan 96
- clickYPercent: varsayılan 8
Bunlar ekranın yüzde koordinatlarıdır. Extension başka yerdeyse website veya koddan değiştirilebilir.
