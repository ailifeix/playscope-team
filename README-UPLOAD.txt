PlayScope Render Upload

Upload these files to the root of the GitHub repo:

- package.json
- outputs/playscope-server.js
- outputs/playscope-prototype.html

Do not upload .env.local. Add API keys in Render Environment Variables instead.

Required Render env vars:
- SERPAPI_KEY
- YOUTUBE_API_KEY
- OPENAI_API_KEY
- GPT_API_BASE_URL
- GPT_MODEL
